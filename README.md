# SmartTax AI Chatbot

## Project Overview

AI-powered tax assistant for freelancers and small businesses.

Tech Stack:

- Backend: FastAPI (Python)
- Frontend: Next.js (TypeScript)
- Database: PostgreSQL
- AI: OpenAI API

---

## Backend Setup

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

---

## Frontend Setup

cd frontend
npm install
npm run dev
